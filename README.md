# koa-jsonapi-bolerplate

Boilerplate based [Koa.js](https://koajs.com/) and JSONAPI v1 specification ([link](https://jsonapi.org/format/))

Example API:

```json
{
  "jsonapi": {
    "version": "1.0.0"
  },
  "data": "Koa.js JSONAPI boilerplate",
  "error": null,
  "meta": {
    "NODE_ENV": null,
    "service_name": "koa-jsonapi-boilerplate"
  }
}
```
